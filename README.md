# LocalUp — Astro + Tailwind + Figma MCP Technical Documentation Pack

This pack is a Figma MCP-first technical documentation framework for building the LocalUp homepage in an Astro + Tailwind project.

It intentionally focuses on implementation structure, Figma extraction, token mapping, component architecture, responsive behavior, and QA.

It does not define final marketing copy or detailed section wireframes.

## Recommended workflow

1. Lock the Figma build target frame.
2. Use Figma MCP to extract exact visual data.
3. Fill the token extraction and component extraction docs.
4. Create Astro/Tailwind foundation files.
5. Implement reusable layout and UI components.
6. Build sections from extracted component patterns.
7. Compare against Figma.
8. Refine inside the real project.

## Core principle

Figma is the visual source of truth.  
This documentation defines how to translate Figma into clean Astro/Tailwind code.
