# Project Overview

## Purpose

This project is an Astro + Tailwind implementation of the LocalUp homepage based on a Figma build target.

The documentation is Figma MCP-first.

## What this pack covers

- Figma MCP extraction
- token extraction
- component extraction
- Astro architecture
- Tailwind/CSS variable mapping
- asset export rules
- implementation phases
- responsive/accessibility
- QA

## What this pack does not cover

- final marketing copy
- detailed section wireframes
- pricing decisions
- proof/testimonial decisions
- business strategy
