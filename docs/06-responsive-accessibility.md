# Responsive and Accessibility Rules

## Responsive

Review widths:

```txt
375px
768px
1024px
1440px
```

General behavior:

- desktop can use split layouts and grids
- tablet should reduce complexity
- mobile should stack sections
- no horizontal overflow
- buttons must stay tappable
- large visuals may simplify on mobile

## Accessibility

- one `h1`
- section titles as `h2`
- card titles as `h3` where useful
- use real links for navigation
- use buttons for actions
- use accessible accordion behavior
- use visible focus states
- use meaningful alt text for informative images
- use empty alt for decorative images
- do not bake important text into images
- check contrast on dark panels and accent pills
